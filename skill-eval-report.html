Key Insights

  • Both skills achieve 100% pass rate across all evals, vs 70% (QA) and 48% (Humanized Tester) without
  • The Button Appearance Test shows the most dramatic difference (100% vs 20%) — without the BIO reference, the agent flags correct VA.gov patterns as bugs
  • Button Focus QA passes equally with and without skill — straightforward button CSS checks don't need domain guidance
  • The Humanized Tester skill provides the biggest lift (+52pp) because accessibility audits require structured methodology the skill provides
  • Without-skill runs tend to use more tool calls (55 vs 37 on full humanized test) but are less accurate — more work, worse results
  • The skills prevent false positives on correct implementations by providing authoritative BIO pattern reference data
